# Hoja de ruta editorial profunda

## Proposito

Este documento convierte el crecimiento del repositorio en un plan concreto. No es una lista abierta de ideas. Es un orden de trabajo recomendado para extender el contenido con profundidad real, minimizando duplicaciones y construyendo una base que haga mas faciles los articulos posteriores.

## Estado actual

Las monografias previstas en esta hoja para `17` a `31` ya existen en el repositorio. A partir de ahora, este documento queda como:

- orden recomendado de lectura profunda
- criterio para futuras revisiones y ampliaciones
- backlog de infraestructura editorial transversal todavia pendiente

## Criterio de orden

El orden optimo no es:

- abrir primero mas escuelas menores
- ni seguir acumulando temas laterales por novedad

El orden optimo es:

1. cerrar primero los grandes problemas transversales que alimentan muchas interpretaciones a la vez
2. pasar luego al frente relativista, de campos, cosmologico y gravitacional
3. desdoblar despues las familias que ya estan bien mapeadas pero todavia demasiado comprimidas
4. construir por ultimo infraestructura editorial de alto valor transversal

## Estado de partida

A fecha actual, el repositorio ya cubre con solidez:

- las grandes familias principales
- Bell, contextualidad y varios no-go theorems
- observador, tiempo, realidad, ontologia, espacio-tiempo e identidad

Lo que falta no es el mapa basico. Lo que falta es la segunda capa profunda.

## Fase 1: Nucleo transversal

Esta fase debe venir primero porque cada una de estas piezas alimenta muchas de las demas.

`docs/articulos/17-decoherencia-bases-preferidas-y-mundo-clasico.md`

Objetivo:

- separar de forma sistematica que hace y que no hace la decoherencia

- Zeh
- Zurek

- Korbicz

- Deutsch

### Prioridad 3

`docs/articulos/19-la-funcion-de-onda-estatuto-ontologico-semantico-y-nomologico.md`

## Fase 2: Frente relativista, de campos y cosmologico

## Regla de trabajo para cada nueva monografia

1. Definir con una frase que problema exacto resuelve el articulo.
2. Reunir primero fuentes primarias y revisiones de calidad.
3. Explicitar que gana y que paga cada familia frente a ese problema.
4. Enlazar siempre con 3 a 5 articulos del repositorio ya existentes.
5. Actualizar `docs/referencias-anotadas.md` y, si hace falta, `docs/bitacora.md`.

## Cierre

La forma mas rapida de inflar un proyecto asi es seguir abriendo temas laterales sin haber estabilizado los grandes nudos conceptuales comunes. La forma mas profunda de hacerlo crecer es la contraria: cerrar primero decoherencia, Born, `psi`, hechos observados, composicionalidad y luego dar el salto a QFT, cosmologia y gravedad cuantica. Esa secuencia no solo produce mas contenido. Produce un corpus mas fuerte, mas util y mejor articulado.
