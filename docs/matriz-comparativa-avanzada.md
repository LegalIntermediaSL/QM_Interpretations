# Matriz comparativa avanzada de interpretaciones cuanticas

## Proposito

Este documento ofrece una comparacion transversal de las principales interpretaciones y programas. No sustituye a los articulos largos del repositorio. Sirve como instrumento de estudio, repaso y cruce conceptual.

Si ya conoces este mapa base y quieres una comparacion con mas ejes, continua en [Matriz comparativa avanzada v2](matriz-comparativa-avanzada-v2.md).

## Como leer esta matriz

- No todas las filas son igualmente decisivas para todos los lectores.
- Algunas casillas simplifican debates internos reales dentro de cada familia.
- La meta no es declarar una ganadora, sino visualizar donde paga y donde gana cada enfoque.

## 1. Matriz general

| Familia | Ontologia tipica | Estatuto de `psi` | Colapso fundamental | Resultados definidos | Probabilidad | Coste principal |
| --- | --- | --- | --- | --- | --- | --- |
| Copenhague | contextual o ambigua | herramienta o estado contextual | ambiguo | si, operativamente | regla operacional | vaguedad del corte |
| Ensemble | conjuntos estadisticos | descriptor de ensembles | no fundamental | si, a nivel de resultados estadisticos | frecuencias | poco dice sobre casos individuales |
| Bohm | particulas mas onda piloto | psi-ontico | no | si | equilibrio cuantico | no localidad explicita |
| Everett | funcion de onda universal y ramas efectivas | psi-ontico | no | si, por ramificacion relativa | problema central de derivacion e interpretacion | multiplicidad de ramas |
| GRW / CSL | estado fisico con dinamica estocastica | psi-ontico | si | si | azar fundamental | nueva dinamica y parametros |
| Historias consistentes | historias decoherentes | depende del marco, pero no meramente instrumental | no | si, dentro de marcos consistentes | probabilidades sobre historias | multiplicidad de marcos |
| Modales | propiedades actuales mas estructura modal | tipicamente psi-ontico o estructural | no | si | probabilidades sobre posibilidades definidas por el estado | seleccion de observables definidos |
| RQM | hechos relativos entre sistemas | relacional | reinterpretado | si, relativos | contextual y relacional | debilita hechos absolutos |
| QBism | experiencias y creencias de agentes | fuertemente psi-epistemico | no, solo actualizacion | si, para cada agente | grados de creencia | ontologia microscopica muy delgada |
| Transaccional | ondas de oferta y confirmacion | realista en algun sentido fisico | variable segun formulacion | si | depende de la formulacion | precision formal disputada |
| TSVF / retrocausal | estados iniciales y finales o condiciones de contorno bidireccionales | variable | variable | si, con tiempo simetrico | depende del marco | ontologia menos estable |
| Superdeterminismo | variables profundas correlacionadas con ajustes | variable | no | si | determinismo profundo | independencia estadistica sacrificada |

## 2. Medicion y resultado

| Familia | Como trata la medicion | Que explica el resultado unico |
| --- | --- | --- |
| Copenhague | como proceso especial o contextualmente definido | colapso operativo o cierre experimental |
| Ensemble | como actualizacion estadistica de un ensemble | la teoria no describe el individuo de la misma forma |
| Bohm | como interaccion cuantica ordinaria | la configuracion real selecciona una rama efectiva |
| Everett | como evolucion unitaria universal | cada rama contiene un observador con resultado definido |
| GRW / CSL | como proceso fisico con inestabilidad macroscopica | la dinamica colapsa el estado |
| Historias consistentes | como parte de una historia dentro de un marco | el marco consistente permite asignacion probabilistica |
| Modales | como actualizacion de posibilidades y actualidades | algunas propiedades son actuales sin colapso |
| RQM | como interaccion entre sistemas | el hecho es relativo al sistema con el que se interactua |
| QBism | como accion del agente y experiencia subsiguiente | el resultado es experiencia del agente y revision coherente |
| Transaccional | como transaccion emisor-absorbente | la transaccion consumada fija el evento |
| TSVF / retrocausal | como evento entre condiciones iniciales y finales | la estructura temporal bidireccional fija el contexto |
| Superdeterminismo | como proceso determinado por condiciones profundas correlacionadas | la correlacion global elimina la necesidad de independencia |

## 3. Localidad, contextualidad y Bell

| Familia | Bell | Contextualidad | Localidad clasica |
| --- | --- | --- | --- |
| Copenhague | acepta el resultado sin restaurar variables ocultas locales | compatible con contextualidad | no recuperada |
| Ensemble | no ofrece restauracion local completa por si sola | depende de la variante | no recuperada claramente |
| Bohm | acepta Bell y paga con no localidad | contextual | abandonada |
| Everett | no restaura localidad clasica de variables ocultas | contextualidad compatible con el formalismo | no recuperada en sentido clasico |
| GRW / CSL | no evade Bell por simple colapso | contextualidad sigue siendo relevante | no recuperada de forma clasica |
| Historias consistentes | acepta la estructura cuantica | fuertemente sensible a incompatibilidad de marcos | no recuperada simplemente |
| Modales | deben convivir con Bell y contextualidad | problema central | no recuperada sin costes |
| RQM | reinterpreta el estatuto de los hechos | compatible con contextualidad | no recuperada como cuadro clasico fuerte |
| QBism | reinterpreta que describe la teoria | contextualidad leida normativamente | no es la meta central |
| Transaccional | intenta acomodar correlaciones con imagen espacio-temporal no clasica | variable | locality fuerte clasica no recuperada sin matices |
| TSVF / retrocausal | puede aliviar parte de la presion no local con condiciones temporales | contextualidad sigue pesando | intenta redistribuir el coste |
| Superdeterminismo | evade la conclusion clasica de Bell rechazando independencia estadistica | compatible con contextualidad, pero no la anula | intenta preservarla a otro precio |

## 4. Probabilidad

| Familia | Que significa la probabilidad |
| --- | --- |
| Copenhague | regla de conexion entre formalismo y resultados experimentales |
| Ensemble | frecuencia en conjuntos preparados de modo similar |
| Bohm | ignorancia sobre configuracion exacta dentro de equilibrio cuantico |
| Everett | problema de auto-localizacion, decision o peso de ramas |
| GRW / CSL | estocasticidad fundamental de la dinamica |
| Historias consistentes | probabilidades sobre historias dentro de un marco consistente |
| Modales | probabilidades sobre propiedades posibles y actualidades |
| RQM | probabilidades relativas a contextos de interaccion |
| QBism | grados de creencia de un agente sujetos a coherencia normativa |
| Transaccional | depende de como se formalice la transaccion |
| TSVF / retrocausal | depende de la lectura de pre y postseleccion |
| Superdeterminismo | puede ser aparente, emergente o estadistica sobre variables profundas |

## 5. Ventajas comparativas

| Familia | Mayor fortaleza |
| --- | --- |
| Copenhague | enorme cercania a la practica fisica real |
| Ensemble | sobriedad empirista y antirrealismo fuerte sobre `psi` |
| Bohm | claridad ontologica y resultados definidos |
| Everett | economia dinamica y universalidad de Schrodinger |
| GRW / CSL | ataque frontal y testable al problema de la medicion |
| Historias consistentes | tratamiento elegante de sistemas cerrados y cosmologicos |
| Modales | intento refinado de equilibrio entre actualidad y posibilidad |
| RQM | critica fuerte a los hechos absolutos independientes |
| QBism | limpieza epistemica y conceptualizacion rigurosa de actualizacion |
| Transaccional | intuicion espacio-temporal sugerente |
| TSVF / retrocausal | relectura profunda del tiempo y la postseleccion |
| Superdeterminismo | reabre Bell desde una salida logicamente permitida |

## 6. Debilidades comparativas

| Familia | Mayor dificultad |
| --- | --- |
| Copenhague | ambiguedad del corte y del colapso |
| Ensemble | falta de ontologia para el caso individual |
| Bohm | no localidad y tension relativista |
| Everett | probabilidad y estatus exacto de las ramas |
| GRW / CSL | nueva dinamica dificil de ajustar y extender |
| Historias consistentes | proliferacion de marcos |
| Modales | regla de asignacion de propiedades poco unica |
| RQM | reconstruccion de intersubjetividad robusta |
| QBism | respuesta ontologica insuficiente para muchos fisicos |
| Transaccional | precision formal y aceptacion desigual |
| TSVF / retrocausal | ontologia dispersa y riesgo de opacidad |
| Superdeterminismo | coste metodologico y falta de formulaciones maduras ampliamente aceptadas |

## 7. Afinidades de lectura

| Si priorizas... | Tiendes a mirar hacia... |
| --- | --- |
| ontologia clara | Bohm, GRW, algunas modales |
| una sola dinamica universal | Everett |
| sobriedad operativa | Copenhague, ensemble, pragmatistas |
| reconstruccion basada en agencia o informacion | QBism, enfoques informacionales |
| reinterpretar hechos sin colapso ni muchos mundos | RQM |
| explorar salidas temporales no convencionales | TSVF, retrocausalidad, transaccional |
| reabrir Bell desde una puerta costosa pero posible | superdeterminismo |

## 8. Preguntas de control

Una buena forma de usar la matriz es intentar responder, para cada familia:

1. Que existe segun esta propuesta.
2. Que cambia respecto al formalismo estandar.
3. Que resuelve explicitamente del problema de la medicion.
4. Que problema nuevo introduce.
5. Cual es su deuda principal con Bell, contextualidad y `psi`.

## 9. Uso recomendado dentro del repositorio

- Lee primero la [guia maestra](guia-interpretaciones-mecanica-cuantica.md).
- Usa esta matriz para comparar familias de un vistazo.
- Ve luego a los articulos largos para descomprimir cada casilla en detalle.

## Cierre

Ninguna matriz sustituye a la argumentacion. Pero una buena matriz obliga a no perder el hilo de comparacion mientras uno entra en textos largos y discusiones especializadas. En un tema tan propenso a mezclar formalismo, filosofia e intuicion fisica, esa disciplina comparativa no es un lujo: es parte de la calidad intelectual del proyecto.
